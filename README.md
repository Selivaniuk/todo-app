## О проекте

Простое приложение для добавления, удаления и выполнения задач, которые сохроняются в localStorage.

Стек: TS + React + SCSS.

Сложность: 1/10

Время выполнения: 5ч.


# Dev

1. Install deps: 
   ```bash
   npm i
   ```

2. Start project:
   ```bash
   npm start
   ```
